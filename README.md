# Projet_Emilie – Site de vente de matériel médical avec devis automatisé

> Projet académique de gestion de projet et développement web (HTML, CSS, JS, PHP/MySQL)  
> **Rôle** : Cheffe de projet | 4 semaines | Équipe de 3

## Description
Projet académique mené en méthode Agile, visant à créer une plateforme de vente dédiée aux associations, avec génération automatique de devis.  
Bien que le projet ne soit pas finalisé, une avancée significative a été réalisée sur la structure front (HTML/CSS) et la conception globale.

## Fonctionnalités implémentées
- Structure HTML sémantique
- Style CSS responsive
- Conception UX/UI (maquettes Figma)

## Technologies utilisées
- **Frontend** : HTML5, CSS3
- **Backend (amorcé)** : PHP, MySQL
- **Outils** : Figma, Canva, draw.io, GitHub, Google Sheets, Discord
